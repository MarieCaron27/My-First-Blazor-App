# My First Blazor App


## 📖 Description

Ce site internet va me permettre de découvrir Blazor et de réaliser différentes manipulations afin 
d'en comprendre les concepts et outils de base de cette technologie.

- Le dossier `code` contient le code source de l'application.
- Dans le dossier `images` se trouvent les images utilisées pour le projet.
- Le dossier lib contient toutes les vues, les controllers, les repositories et les constantes du projet.
- Le fichier LICENSE à la racine du dépôt indique les termes et conditions de la licence sous laquelle le code source est distribué.
